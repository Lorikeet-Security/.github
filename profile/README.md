Welcome to the Lorikeet Security / Parrot CTFs GitHub — home to open-source tools, CTF challenge backends, recon scripts, and security resources built by our team. Parrot CTFs is the training and events arm of Lorikeet Security, offering hands-on ethical hacking labs, Capture the Flag competitions, and professional CTF event hosting for individuals, teams, and enterprises.

---

### Introduction

Parrot CTFs delivers real-world cybersecurity training through interactive CTF challenges, vulnerable lab machines, and structured academic content across web security, active directory, cryptography, and forensics. Our challenges are crafted to mirror genuine attack scenarios — giving practitioners meaningful reps in a safe, controlled environment.

Parrot CTFs is also a division of **Lorikeet Security**, which provides penetration testing, attack surface management (ASM), cyber awareness training, phishing simulations, and professional CTF event hosting for organizations of all sizes.

---

### What's in This Repository

Here's a snapshot of our current public repos:

- **[PCTFS-MICROLABS](https://github.com/Parrot-CTFs/PCTFS-MICROLABS)** — A free, standalone web application for hands-on cybersecurity skills practice. No subscription required.
- **[PCTFS_Crypto_Scanner](https://github.com/Parrot-CTFs/PCTFS_Crypto_scanner)** — Analyze any local workspace or GitHub repository for cryptographic primitives, risky implementations, and secret handling.
- **[install-diy-local-lab](https://github.com/Parrot-CTFs/install-diy-local-lab)** — Tired of non-stop subscriptions? Deploy a local, scaled-down version of popular vulnerable lab environments.
- **[Grpc_TradingServer](https://github.com/Parrot-CTFs/Grpc_TradingServer)** — Realtime price change server built with Kafka. Powers the *Wall Street Hijack* CTF challenge.
- **[parrot-recon](https://github.com/Parrot-CTFs/parrot-recon)** — Recon automation tooling for bug bounty hunters.
- **[MobileApp-Pentest-Cheatsheet](https://github.com/Parrot-CTFs/MobileApp-Pentest-Cheatsheet)** — A concise, high-value reference for mobile application penetration testing topics.
- **[OWASP-Testing-Checklist](https://github.com/Parrot-CTFs/OWASP-Testing-Checklist)** — An Excel-based OWASP web application security testing checklist for tracking test case status.
- **[pentest-automation](https://github.com/Parrot-CTFs/pentest-automation)** — A growing collection of penetration testing automation scripts.
- **[Parrot_CTFs-Layan-gtk-theme](https://github.com/Parrot-CTFs/Parrot_CTFs-Layan-gtk-theme)** — Parrot CTFs-branded GTK theme for AthenaOS.

### Contribution Guidelines

We welcome community involvement! Here's how to participate:

- **Report Bugs**: Open an issue on the relevant repository's GitHub Issues page or email us directly.
- **Submit PRs**: Fork any public repo, make your changes, and submit a pull request. Please keep commits focused and include context in your PR description.
- **Sponsor**: Support our open-source work via [GitHub Sponsors](https://github.com/sponsors/Parrot-CTFs).
